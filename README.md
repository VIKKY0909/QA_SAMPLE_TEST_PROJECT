# QA Testing Project for Demo Website

## Project Overview
This repository contains QA testing artifacts for the Demo Website (https://www.bstackdemo.com/). The project includes a comprehensive test plan, detailed test cases, and a bug report generated during testing.

## Key Features
- **Test Plan:** Outlines the testing strategy, scope, objectives, and schedule.
- **Test Cases:** 34 well-documented test cases covering functional, boundary, negative, and UI/UX tests.
- **Bug Report:** Documents critical and high-priority bugs with detailed steps and severity levels.

## Tools Used
- **BrowserStack:** Cross-browser testing.

- **Microsoft Word and Excel:** Documentation and reporting.
